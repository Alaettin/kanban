DTI Connector API

API Documentation

Public REST API for the DTI Connector.

Base URL

Authentication

Your personal API key (UUID) is part of the base URL. Generate and save your key on the Settings page of the DTI Connector app. Each API key is unique per user — your data is fully isolated.

Endpoints

GET /Product/ids List all item IDs

Returns an array of all asset (item) IDs.

Response 200 OK
["CBL001", "CBL002", "DISH001", "DISH002"]
Error 401 Unauthorized
{ "error": "Invalid API key" }
GET /Product/hierarchies List hierarchies

Returns an empty array. Reserved for future use.

Response 200 OK
[]
Error 401 Unauthorized
{ "error": "Invalid API key" }
GET /Product/hierarchy/levels List hierarchy levels

Returns all defined hierarchy levels, ordered by level number.

Response 200 OK
[
  { "level": 1, "name": "ProductType" },
  { "level": 2, "name": "Manufacturer" },
  { "level": 3, "name": "Series" }
]
Error 401 Unauthorized
{ "error": "Invalid API key" }
GET /model List model datapoints

Returns all defined datapoints with their ID, name, and type (0 = Property, 1 = File).

Response 200 OK
[
  { "id": "manufacturer", "name": "Manufacturer", "type": 0 },
  { "id": "datasheet", "name": "Datasheet", "type": 1 }
]
Error 401 Unauthorized
{ "error": "Invalid API key" }
GET /Product/{itemId}/hierarchy Get hierarchy values for an asset

Returns the hierarchy level values for a specific asset. Each level shows the configured value as name.

Response 200 OK
[
  { "level": 1, "name": "Electric engineering" },
  { "level": 2, "name": "Cable, wire" },
  { "level": 3, "name": "Energy cable" }
]
Error 401 Unauthorized
{ "error": "Invalid API key" }
Error 404 Not Found
{ "error": "Asset not found" }
POST /Product/{itemId}/values Get property values for an asset

Returns property values for a specific asset. Use the request body to filter by language and/or property IDs. Send empty arrays to get all values.

Request Body
{
  "propertiesWithLanguage": {
    "languages": ["en", "de"],
    "propertyIds": ["dp.color"]
  },
  "propertiesWithoutLanguage": {
    "propertyIds": ["dp.weight"]
  }
}
