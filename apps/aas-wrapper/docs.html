AAS Repository Proxy — API

API Documentation

AAS Repository API nach IDTA-01002 V3.1 Spezifikation.

Base URL

Asset Administration Shell Endpoints

GET /shells CACHED Alle Shells auflisten

Gibt alle Asset Administration Shells als persistierte Liste zurück. Die Daten werden über die Config-Seite aufgebaut und in der Datenbank gespeichert.

Response 200 OK
{
  "paging_metadata": { "cursor": "" },
  "result": [
    {
      "idShort": "ExampleShell",
      "id": "https://example.com/aas/ExampleShell#1",
      "assetInformation": {
        "assetKind": "Instance",
        "globalAssetId": "https://example.com/asset/1",
        "assetType": "IRI"
      },
      "submodels": [
        {
          "type": "ModelReference",
          "keys": [{ "type": "Submodel", "value": "https://example.com/submodel/1" }]
        }
      ]
    }
  ]
}
GET /shells/{aasIdentifier} Shell abrufen

Ruft eine einzelne Asset Administration Shell anhand ihres base64url-codierten Identifiers ab.

Response 200 OK
{
  "idShort": "ExampleShell",
  "id": "https://example.com/aas/ExampleShell#1",
  "assetInformation": { "assetKind": "Instance", "globalAssetId": "https://example.com/asset/1" },
  "submodels": [...]
}
Error 502 Bad Gateway
{ "error": "PROXY_ERROR", "message": "HTTP 404" }
GET /shells/{aasIdentifier}/asset-information Asset-Information

Gibt die Asset-Information (assetKind, globalAssetId, assetType) einer Shell zurück.

Response 200 OK
{
  "assetKind": "Instance",
  "globalAssetId": "https://viewer-dogan.dti.neoception.dev/300128",
  "assetType": "IRI"
}
GET /shells/{aasIdentifier}/asset-information/thumbnail Thumbnail

Gibt das Thumbnail-Bild der Asset-Information zurück (binär).

GET /shells/{aasIdentifier}/submodel-refs Submodel-Referenzen

Listet alle Submodel-Referenzen einer Shell auf. Jede Referenz enthält den Typ und den Schlüssel zum Submodel.

Response 200 OK
{
  "result": [
    {
      "type": "ModelReference",
      "keys": [{ "type": "Submodel", "value": "https://..." }]
    }
  ]
}
GET /shells/{aasIdentifier}/submodels/{submodelIdentifier} Shell-Submodel abrufen

Ruft ein spezifisches Submodel einer Shell ab. Beide Identifier müssen base64url-encoded sein.

Response 200 OK
{
  "idShort": "Identification",
  "id": "https://...",
  "submodelElements": [...]
}
GET /serialization CACHED AAS Environment JSON

Gibt das gesamte AAS Environment als JSON zurück. Enthält alle gecachten Shells, Submodels und Concept Descriptions. Wird aus dem lokalen Cache zusammengebaut.

Response 200 OK
{
  "assetAdministrationShells": [
    {
      "idShort": "ExampleShell",
      "id": "https://example.com/aas/1",
      "assetInformation": { "assetKind": "Instance", "globalAssetId": "..." },
      "submodels": [...]
    }
  ],
  "submodels": [
    {
      "idShort": "Identification",
      "id": "https://example.com/submodel/1",
      "submodelElements": [...]
    }
  ],
  "conceptDescriptions": []
}

Submodel Endpoints

GET /submodels CACHED Alle Submodels auflisten

Gibt alle gecachten Submodels zurück. Die Submodels werden beim Daten-Refresh automatisch aus den Shell-Referenzen extrahiert und abgerufen.

Response 200 OK
{
  "paging_metadata": { "cursor": "" },
  "result": [
    {
      "idShort": "Identification",
      "id": "https://example.com/submodel/1",
      "kind": "Instance",
      "submodelElements": [
        {
          "idShort": "ManufacturerName",
          "modelType": "Property",
          "valueType": "xs:string",
          "value": "Example Corp"
        }
      ]
    }
  ]
}
GET /submodels/{submodelIdentifier} Submodel direkt abrufen

Ruft ein Submodel direkt über seinen base64url-codierten Identifier ab (ohne Shell-Kontext).

Response 200 OK
{
  "idShort": "Identification",
  "id": "https://...",
  "submodelElements": [...]
}
GET /submodels/{submodelIdentifier}/submodel-elements/{idShortPath} Submodel-Element

Greift auf ein bestimmtes Element innerhalb eines Submodels zu. Der idShortPath verwendet Punkt-Notation (z.B. Identification.Country).

Response 200 OK
{
  "idShort": "Country",
  "modelType": "Property",
  "valueType": "xs:string",
  "value": "DE"
}
GET /submodels/{submodelIdentifier}/submodel-elements/{idShortPath}/attachment Element-Attachment

Lädt das Attachment (Datei) eines Submodel-Elements herunter. Gibt binäre Daten zurück.

Utility Endpoints

GET /description API-Beschreibung

Gibt die Self-Description des AAS Repositories zurück. Enthält unterstützte Profile und Interfaces.
